*nix系统解压神器dtrx

先丢上网址，心急的同学可以直接跑到项目主页去看原版描述。一句话介绍就是一条命令解压所有格式的压缩包。就像他自己的描述——Intelligent archive extraction。

下载、安装和介绍猛戳这里:

http://brettcsmith.org/2007/dtrx/

好了，之后我是随口叨叨几句。

这个玩意么是python写的，照道理呢windows也可以用的，可惜windows不太会用所以就没试过，有兴趣可以自己搞搞。当然也许会有大牛跑出来说，老子tar -zcvf 打的非常熟练，我不要用那么低级的东西。。。那好吧，随便。反正我觉得不管啥压缩包都打一个dtrx xxx.xxx 挺方便的。

安装过程么，很简单，下载解压后用root一句命令搞定 

python setup.py install –prefix=/usr/local
然后，享受吧少年。
