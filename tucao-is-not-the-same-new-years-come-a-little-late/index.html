吐槽一样的新年计划是不是来的有点晚

很无厘头的选择了一个那么长而且没有什么特别含义的文字来作为今年我的新年计划的题目。恩，其实我是不习惯列表那样的写一个新年计划，然后在年末来看看自己完成了多少什么的，所以我的新年计划一如既往的是混乱无比的碎碎念。

先说说最近的事情吧，这也是为什么我到那么晚才写这篇文章的部分原因。最近最大的事情就是这周末就要去魔都，不出意外下周就会开始工作了，虽然一开始是以实习生的身份，不过也没有什么可以懈怠退缩的理由罢。我对朋友总是说，我有入职前焦虑症，所以我最近焦虑着，常常做事很混乱，容易忘记一些刚刚想到的事情，不过还好，Evernote 的存在让我很快走出这种窘迫的情景。不过，一如既往的“除夕情绪低落症候群”在我身上又非常准时的出现了，每年除夕的时候我的情绪都是不怎么高扬的，总之算是不怎么愉快的，算是一个莫名其妙的规律，所以今年一如既往，除夕还是不那么的舒服，对人对事反映都是不那么好。不过过完初七感觉瞬间就好多了，人格情绪都正常起来，这倒是让我欣慰不已。

转说工作的事情，放假前信誓旦旦的说要对自己二次充电，这个算是勉勉强强完成了一部分。无论是Android还是Java基础都算是学习了一些新的东西，不过我再次感觉到，其实做Android开发，Java语言的能力还是很重要的，所以现在正在抽空认证研究一本书《Java 语言程序设计》  其实就是《Introduction to Java Programming》的中文版本罢了，讲述一些语言的高级用法用来提高还是不错的。技术上的事情和考试不同，对于国内的考试，高考，中考抑或后来大学的各种水的一踏糊涂的考试，除去作弊这类非常规方法，其实解决方法都一样——题海战术。不知道这算不算是中国特色，记得很多东西在国内弄到最后都成为了题海战术，买油翁的一句：“无他，唯手熟耳”还真是贴切。不过对于作技术，题海自然是无用的，私认为大量的实践和探索才是提高的最好办法，所以对工作我还是异常的期待，纵使会遇到许多意料之外的大挑战，但是学到的那些大抵也是让我终生受益的罢。

说完工作的事情，扯扯摄影。最近很少拍片，不是没心情，而是如上面说的情绪不稳定，感觉无法安静的去拍摄。我一直固执的认为，摄影是一件很私人的事情，拍摄，观察，拍摄，观察，拍摄，观察大抵就是这么一个循环，如果没有一个很好的心境是无法通过照片折射出内心的姿态的，我估且那么的想着。所以打包打包摄影器材，打算去了魔都后看看有没有机会，有空扫扫街，抑或拍拍一些小玩意也是其乐无穷。

关于情感，情感的问题其实要求是很平实的。我不希望有什么轰轰烈烈，也不希望什么跌宕起伏，太折腾。如果平平淡淡的稳定，那固然是最好不过的了。所以，关于情感，做好自己的角色，稳稳当当的前进。

关于数码设备，终于用Galaxy Tab 10.1换掉了和我水土不服的iPad 2,总之对于iOS的产品一直让我有透不过气的感觉。重新开始用Kindle 3，放了一些比较淡的书进去，因为可见的未来几天，我会有比较空荡的时间，拿来看看书也是个不错的选择罢。

最后还是自私的祝福一下自己，希望12年自己一切顺利。
