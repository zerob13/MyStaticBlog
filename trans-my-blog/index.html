迁移博客

昨天在微博上看到贾兄分享了一个新的博客环境—— Ghost ，好奇的研究下了，然后就决定抛弃 Wordpress 迁移过来。 新环境是 Node.js + sqlite + nginx 项目地址如下：

Ghost

什么？你问我为什么要迁移过来？有什么优势？我就是觉得这个玩意够轻够简单，又够强大，而且也想折腾一下 nginx 下面同时跑 php 和 node.js。

运气不错配置很简单，反向代理的内容简单学了下 nginx 的语法也搞定了。

所以发一篇博客试试，这个系统支持Markdown语法，这点不错，写博客轻松多了，而且用鼠标的次数也少了很多。

总体来说这个系统还很不成熟，比如不能搜索文章，比如文章管理页面非常简单很多功能都还没有，但这都不是事儿，总会好的。并且编写的时候实时看到最终效果这个也是一个 很不错的Feature。

目前一切很爽。
