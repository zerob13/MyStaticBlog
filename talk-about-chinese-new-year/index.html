侃大年

2014-02-01

  蛇年的年三十，在大家准备年夜饭，准备吐槽春晚然后喜迎马年的时候，我正在电脑前，亦步亦趋的学写 Go 语言。倒不是自己有多努力，也不是加班什么的，只是觉得既然过年总还是要做点自己喜欢的事情。

  所以，还是觉得写会儿代码比较容易在这样喜庆的吵闹环境中找到一点儿平静。活着现在的社会上，做自己大抵是一个非常奢侈的奢侈品。就不说正常社交中的伪装和社交礼仪了，很多的时候，为了保护弱小的自己，不得不套入一个大大的壳中，这也不免是一个很悲哀的事情。

  过年，总是要去反省过去，展望未来。诚然，对过去一年的总结我已经在 2013 年的 12 月写完了所以也不打算再次重复一遍。不过对于新年的展望倒是一直没个准信，多半也是因为自己觉得计划赶不上变化，长期计划一般都是扯淡，基本没法实现，不信可以翻翻每年计划的看书啊，学习啊啥的。所以还是说说一些短期的计划来的靠谱点。

  短期来说，打算学习一些后段开发的技术，比如 Go 语言，比如 Nginx 的一些东西，这方面的知识的囤积和积累我一直觉得未来几年会用得上。毕竟移动互联网时代还是互联网时代，后端技术依然是非常重要的。而且终端数的大量增加，从上网行为变成在网行为，对并发和实时的性能要求也会非常之高，所以才会选择 Go 语言。几年前选择跟着 Google 走学了 Android，算是走对了；希望这次 Google 也别让我失望:P

   另外，也希望能在更基础的技术上面有一些突破，温故而知新，捡回荒废了好久的 C++ 算是近几个月比较重要的一个事情。不过有一个好事儿是捡回了荒废了好久的 Vim 之前做 Android 基本都去 Eclipse 和 Android Studio 了，终于现在学后段的时候可以回归 VIM 算是我很欣慰的一个事情。

  要增加的知识点当然不止这么一些，不过如果这篇文字就是用来把这些计划排排站的话，那就基本没啥意义了。还是要谈谈一些其他的事儿的。主要还是想说说苦难和选择。

  之前在知乎收到不少学弟学妹的私信来询问各种就业的想法，也断断续续的谈了一些看法。大抵不外乎做自己喜欢的事情，坚持自己的兴趣之类的话语。看似空洞，其实我是觉得不管去做什么都会不可避免的遇到一大堆的困难和痛苦，那还不如为自己喜欢的事情拼搏一把，至少过程会好受一点。人生在世，真的就是来受苦受难的，有的人是肉体上，有的人是物质上，也有些人是精神上。我一直是一个很敏感而且略带神经质的人，有时候一些压力会让我整晚睡不好觉。但是后来想想，还是要坦然去面对，遇到问题，去面对才有希望解决，逃避只会把问题养肥了。就像学习一个新知识，一开始总是不停的排斥，大吼着”WTF“来试图逃避，但是硬着头皮上去了又是一片天空。

  春晚一如既往的被吐槽，看了一部分我就不看了，继续回房间学 Go 。发现 Go 的变量类型声明是后置的，和C-like 的语言完全不同，顿时有点抵触，不过继续看下去，看了 Go 的面向对象，接口，函数重写等等，突然觉得还是挺有趣，渐渐的也被这语言略略的洗脑，不过目前还是处于一个混乱期，接受新事物的过程总是比较痛苦，但愿结局能好一些。

  每到过节，我的情绪总会不由自主的往下掉，大抵是太过于矫情了。每当万千神仙歆享着人间的香烟的时候，我总会忍不住想起一些人还在外面风餐露宿，当然这或许只是我的一种莫须有的圣母情结罢了。抑或是我对于生活的残酷无奈的映射，不管如何，我总是不太喜欢这样明显的对比。但似乎也无法去改变什么，所以还是想想专注自己身边的事情，就不要地命海心了。

  年前霍金似乎发了新论文，貌似说黑洞的视界是不存在的之类的内容。总之，上一次看这方面的东西似乎还是在高中的时候看《时间简史》看的不亦乐乎。想起来自己曾经也是如此喜欢物理学，喜欢看一写物理方面的科普文字。现在似乎到除了计算机啥都不看了，不知道是好是坏，不过我还是挺喜欢看各种电影，算是不错的休闲。

  对于旅游，着实至今没有太大的兴趣。总觉得是一个烧钱找麻烦的事儿。要是陪妹子出门到也罢了，总还有个妞可以看看。自个儿来个说走就走我总觉得有点二。当然，也有可能只是我太宅了。

  摄影，算是怀念许久的老伙伴。很想好好拍拍，可惜各种原因，不仅仅是时间不允许，还有脑力有限很多时候真的没有精力去思考。其实摄影和拍照的区别对我来说就是一个是通过大脑思考去创作，另一个则是通过身体反射来拍一张凑合的照片。所以最后的最后，丢上一张回家时候的虹桥火车站的HDR，算是作为这篇扯犊子的文章的结尾。

来，和我一起念，独自～扯犊子～滚犊子～王八犊子～
