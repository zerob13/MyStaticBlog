第一天

今天是大三的第一天正式上课。 两堂课，共计5节。一个是《算法分析与设计》一个是《计算机图形学》 都比较轻松，主要是课程是入门型的，所以问题不大，基本上就可以打酱油的过去~ 中午又一次见到了@bhuztez,一起吃了饭，聊了很多东西。 包括操蛋的电x的闪讯，技术，python，android，gnome 3，gtk，qt,opengl等等。 哦，下午有个小插曲，就是淘宝给我送来了口碑卡，囧，绑定了一下，有空可以去玩玩看。 然后就是肖磊的测光卡到了，我弄个一个过来，挺好的。现在玩胶片可以用用。 晚上回到寝室，先是把自己电脑的Spaces开启了，然后设定了4个工作区，加上我本来就是双屏，就直接变成了8桌面。。。 然后把工作分工了一下，一个桌面一个任务，专注的做，果然效率有所提升（当然也可能是我的心理作用） 之后，试了试opengl，今天很搞笑，中午的时候还和bh聊天说用opengl在android上做点什么，下午的图形学居然就是讲opengl。 当然，下午讲的那个档次就比较那个啥了。。。不过没是，回来还是在xcode里面跑了跑书上的范例，恩，mac内置了opengl还是很赞的，什么都不用设置，直接写好代码跑就好。果然是M$无法比的，哈哈哈。接着把随书附带的代码范例都拷贝到了电脑里面，恩，可以玩玩，没问题的。 ———————————————————————————————————————— 完成了opengl的helloworld，我选择折腾一个很折腾的事情，就是换虚拟机，过程乱七八糟的艰辛，结果是一塌糊涂的失望，囧。略过不提。 这里可以附上在mac下安装qemu的方法，反正我是都成功了，只是速度不尽人意，估计是没有kqemu优化的关系，因为mac不支持。 安装很简单，shell里面port一下就搞定了。 $ sudo port install qemu 然后是创建虚拟硬盘 $ qemu-img create -f qcow windows.img 10G 接着是从镜像安装windows $ qemu -localtime -cdrom xp.iso -m 1024 -boot d windows.img 万事搞定，安装完毕就可以玩xp了，启动这个虚拟机也相当容易，就一句话 $ qemu -hda windows.img 导出了军训的片，发给肖同学了，这样今天的事情应该都搞定了。 洗个澡，把课表导入到google calendar里面，准备开始正常的大学上课生涯~ 晚安，地球的人们~
