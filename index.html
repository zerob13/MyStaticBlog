高考作文三篇 2014-06-09

又到了每年高考结束后的日子，许多毕业生开始大量涌入各种游戏坑的不亦乐乎。所以，选了三个作文题，我也写了一把，保持年轻，保持愚蠢。

文章基本都是温性感人风格，大家可以随便感受一下。

2014上海卷

根据以下材料，自选角度，自拟题目，写一篇不少于800字的文章(不要写成诗歌) 　　你可以选择穿越沙漠的道路和方式，所以你是自由的；你必须穿越这片沙漠，所以你又是不自由的。

镜

    盛夏午后的洗手间，炙热，潮湿弥漫着清洁剂的味道。镜子里可以看到全貌，五个蹲坑，一条小便槽还有几个龙头西索西索的流着水。云层很厚，低，黑，暴雨前夕。     下午两点二十分。     小明在厕所里发现了不得了。     小明在镜子后面放了另外一面镜子，小明发现在镜子里的水变慢了，他自己也变慢了。与其说变慢了，更确切的是离散了。哦，感谢伟大的必修课，小明懂得离散的概念。     小明用龙头的水洗了把脸，惊奇的看到甩出来的水滴断断续续的飞出去，似乎还出现了诡异的形变。     小明突然明白了。他感到不可思议，但工科狗的头脑让他立刻明白排除所有可能后，剩下的就是真相。     恩，世界是模拟的。     虽然模拟的无比真实，大抵是用了恒星级别的量子计算，但在镜面和镜面无限递归的时候还是露馅了。原来大家都不过是既定的程序，去你妈的freewill,小明胸口起伏不定，大口喘气，照道理来说不可能没人发现。     为什么没人揭穿？     不，我要为世界负责。小明想着各种方法去揭穿真相。我要战斗，我要为了布尔什维克的胜利。     可揭穿了，谁都不自由。不揭穿了，大抵还是可以选择程序执行的周期罢。小明开始颤抖，似乎自己做了一个可怕的决定。可怜的小明，中分的小明，背井离乡的小明，毕业了找不到工作的小明，女朋友跟着富二代跑了的小明。     小明开始愤怒，似乎生命的一切不如意都是由于自己身在一个虚构的世界里，一切都不过是既定的程序。小明的心啪啪啪的跳动，似乎找不到自己，小明有些难受，就像被狗熊甩了一脸那么闷。     “娘西匹，都是假的。”     小明又洗了把脸，突然觉得可笑起来。想起那些为了争一个职位在面试的时候无所不为的同学，想起和富二代跑了的黑木耳，最后不过是数据库里面的一个字段而已。小明突然觉得无比喜感。     “臭傻x。”小明心中默念。     这时的小明已经不是那个小明，小明感到无比舒畅，身体里的血液波澜壮阔的敲打着血管壁，细胞中的线粒体都唱起了亚克西。一种生命和谐的大喜悦充斥了全身。     小明做了个充满镜子的球，蔑笑着走了进去，再也没出来过。

纪念昨天君 2014-05-25

死胖子你最近很久没写东西了啊。

是啊，很久没写了。
死胖子有想法写点啥么？ 大抵需要一点仰望星空的灵感吧。 负分滚粗。

    好久没有写点东西了，再不写对不起服务器的租金，对不起域名的租金，对不起那几十个的订阅用户了。哦，其实只有十几个。     坐车上下班的时候，体内常常有声音响起，督促我是否该写点什么了。我想大抵我是应该写点什么了，但却又不知从何说起，说点什么。正能量？太无趣，鸡汤虽好，容易尿酸高。负能量？太简单，我的脸就是充满了负能量的喜感，贴个自拍就完事了。点评点科技圈？太麻烦，而且这方面的观点和天涯骂战一样，谁也不服谁，就像我不喜欢老罗，但我挺喜欢锤子，我既没有跑去喷罗胖子，同时也默默的订了一台锤子的 T1 。这样就好，这样就好。     既然扯倒了锤子，这算是我今年看下来比较又欲望买的东西了。我喜欢玻璃比例大的手机，比如 Nexus 4 ，现在我还在用，非常喜欢。另外，捐钱给 OpenSSL 这事儿做的漂亮，就冲着这个也应该买个他们的手机，不管是否作秀，就算作秀也希望以后各种厂能多作秀几次。也算是推动人类计算机事业发展了。     最近看着自己的 GitHub 提交记录，惨不忍睹，空空如也。着实觉得虚度了许多光阴，但又不能为了好看而瞎提交一些东西。所以想想为了充实业余生活，还是做点别的事情。然后就跑去翻译了一些外文的技术文章，丢给伯乐在线的 Android 开发小组。倒也有了两篇上了，不由自己觉得让世界变得美好了一点。其中一篇比较有意思，是写一个自己组建的 Android 开发框架，用了一堆第三方的库，但是总体效果很不错，有兴趣的同学可以看一下，点这里。个人建议， SnappyDB 和 EventBus 这两个比较有意思一点，其他的两个风格化太厉害，我个人并不推荐。     人生在世，扯扯犊子乃是一大乐事，当然码农锻炼身体也是必不可少都。每周末都会去打打球，算是最近比较好的一个事情，后续有机会还可以拉几个基友游泳，随缘。 最后送上逗逼视频一段，猛击这里 查看。

爬行动物 2014-04-26

    每天上下班，猫着腰在城市最繁忙的爬行动物的角落呆着。边上充斥着黑木耳亦或是类似自己的傻伯夷，聒噪或木然。
    有时候一朵黑木耳下去，两只杀马特回来；一只傻伯夷离去，几分紫葡萄出现。来来去去，大抵就是他的新陈代谢，没有谁留下什么，就算是对面那个搂着一朵六风黑木耳的死胖子无敌的体臭也在他离去后的几秒中散去。

聊聊我的Vim配置文件 2014-03-31

    程序员用的最多的编辑器莫过于 Vim 和 Emacs 了，由于我是习惯用 Vim 的一个人，也用了几年的 Vim 最近重新整理了自己的配置文件，加上了详细的注释，丢到了 GitHub 上。主要支持的开发环境是 C/C++/JS/HTML/CSS/GO ,以及增加了 VIM-TODO 工具的支持

先丢效果图

安装方法：

$ git clone https://github.com/zerob13/vimrc.git  ~/.vim
$ git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
$ cp ~/.vim/.vimrc ~/.vimrc
$ vim
:BundleInstall

也可以直接访问 GitHub 获取代码

我的Vimrc如下:

"关闭vi的一致性模式 避免以前版本的一些Bug和局限
set nocompatible
"配置backspace键工作方式
set backspace=indent,eol,start
"显示行号
set number
"设置在编辑过程中右下角显示光标的行列信息
set ruler
"在状态栏显示正在输入的命令
set showcmd

"设置历史记录条数
set history=1000

"设置取消备份 禁止临时文件生成
set nobackup
set noswapfile
"设置匹配模式 类似当输入一个左括号时会匹配相应的那个右括号

"设置C/C++方式自动对齐

"开启语法高亮功能

"指定配色方案为256色

"设置搜索时忽略大小写

"设置在Vim中可以使用鼠标 防止在Linux终端下无法拷贝

"设置Tab宽度

"新建.c,.h,.sh,.java,py文件，自动插入文件头

""定义函数SetTitle，自动插入文件头
func SetTitle()
    "如果文件类型为.sh文件
    if &filetype == 'sh'
        call setline(1,"\#########################################################################")
        call append(line("."), "\# File Name: ".expand("%"))
        call append(line(".")+1, "\# Author: zerob13")
        call append(line(".")+2, "\# mail: zerob13@gmail.com")
        call append(line(".")+3, "\# Created Time: ".strftime("%c"))
        call append(line(".")+4, "\#########################################################################")
        call append(line(".")+5, "\#!/bin/bash")
        call append(line(".")+6, "")
    elseif &filetype == 'python'
        call setline(1,"\#!/usr/bin/env python")
        call append(line("."),"\#-*- coding: utf-8 -*- ")
        call append(line(".")+1,"\#Author:  zerob13 <zerob13@gmail.com>")
        call append(line(".")+2,"")
    else
        call setline(1, "/*************************************************************************")
        call append(line("."), "    > File Name: ".expand("%"))
        call append(line(".")+1, "    > Author: zerob13")
        call append(line(".")+2, "    > Mail: zerob13@gmail.com ")
        call append(line(".")+3, "    > Created Time: ".strftime("%c"))
        call append(line(".")+4, " ************************************************************************/")
        call append(line(".")+5, "")
    endif
    if &filetype == 'cpp'
        call append(line(".")+6, "#include<iostream>")
        call append(line(".")+7, "using namespace std;")
        call append(line(".")+8, "")
    endif
    if &filetype == 'c'
        call append(line(".")+6, "#include<stdio.h>")
        call append(line(".")+7, "")
    endif
    "新建文件后，自动定位到文件末尾
    autocmd BufNewFile * normal G
endfunc
