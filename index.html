镜

    盛夏午后的洗手间，炙热，潮湿弥漫着清洁剂的味道。镜子里可以看到全貌，五个蹲坑，一条小便槽还有几个龙头西索西索的流着水。云层很厚，低，黑，暴雨前夕。     下午两点二十分。     小明在厕所里发现了不得了。     小明在镜子后面放了另外一面镜子，小明发现在镜子里的水变慢了，他自己也变慢了。与其说变慢了，更确切的是离散了。哦，感谢伟大的必修课，小明懂得离散的概念。     小明用龙头的水洗了把脸，惊奇的看到甩出来的水滴断断续续的飞出去，似乎还出现了诡异的形变。     小明突然明白了。他感到不可思议，但工科狗的头脑让他立刻明白排除所有可能后，剩下的就是真相。     恩，世界是模拟的。     虽然模拟的无比真实，大抵是用了恒星级别的量子计算，但在镜面和镜面无限递归的时候还是露馅了。原来大家都不过是既定的程序，去你妈的freewill,小明胸口起伏不定，大口喘气，照道理来说不可能没人发现。     为什么没人揭穿？     不，我要为世界负责。小明想着各种方法去揭穿真相。我要战斗，我要为了布尔什维克的胜利。     可揭穿了，谁都不自由。不揭穿了，大抵还是可以选择程序执行的周期罢。小明开始颤抖，似乎自己做了一个可怕的决定。可怜的小明，中分的小明，背井离乡的小明，毕业了找不到工作的小明，女朋友跟着富二代跑了的小明。     小明开始愤怒，似乎生命的一切不如意都是由于自己身在一个虚构的世界里，一切都不过是既定的程序。小明的心啪啪啪的跳动，似乎找不到自己，小明有些难受，就像被狗熊甩了一脸那么闷。     “娘西匹，都是假的。”     小明又洗了把脸，突然觉得可笑起来。想起那些为了争一个职位在面试的时候无所不为的同学，想起和富二代跑了的黑木耳，最后不过是数据库里面的一个字段而已。小明突然觉得无比喜感。     “臭傻x。”小明心中默念。     这时的小明已经不是那个小明，小明感到无比舒畅，身体里的血液波澜壮阔的敲打着血管壁，细胞中的线粒体都唱起了亚克西。一种生命和谐的大喜悦充斥了全身。     小明做了个充满镜子的球，蔑笑着走了进去，再也没出来过。

先丢效果图

安装方法：

$ git clone https://github.com/zerob13/vimrc.git  ~/.vim
$ git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
$ cp ~/.vim/.vimrc ~/.vimrc
$ vim
:BundleInstall

也可以直接访问 GitHub 获取代码

""定义函数SetTitle，自动插入文件头
func SetTitle()
    "如果文件类型为.sh文件
    if &filetype == 'sh'
        call setline(1,"\#########################################################################")
        call append(line("."), "\# File Name: ".expand("%"))
        call append(line(".")+1, "\# Author: zerob13")
        call append(line(".")+2, "\# mail: zerob13@gmail.com")
        call append(line(".")+3, "\# Created Time: ".strftime("%c"))
        call append(line(".")+4, "\#########################################################################")
        call append(line(".")+5, "\#!/bin/bash")
        call append(line(".")+6, "")
    elseif &filetype == 'python'
        call setline(1,"\#!/usr/bin/env python")
        call append(line("."),"\#-*- coding: utf-8 -*- ")
        call append(line(".")+1,"\#Author:  zerob13 <zerob13@gmail.com>")
        call append(line(".")+2,"")
    else
        call setline(1, "/*************************************************************************")
        call append(line("."), "    > File Name: ".expand("%"))
        call append(line(".")+1, "    > Author: zerob13")
        call append(line(".")+2, "    > Mail: zerob13@gmail.com ")
        call append(line(".")+3, "    > Created Time: ".strftime("%c"))
        call append(line(".")+4, " ************************************************************************/")
        call append(line(".")+5, "")
    endif
    if &filetype == 'cpp'
        call append(line(".")+6, "#include<iostream>")
        call append(line(".")+7, "using namespace std;")
        call append(line(".")+8, "")
    endif
    if &filetype == 'c'
        call append(line(".")+6, "#include<stdio.h>")
        call append(line(".")+7, "")
    endif
    "新建文件后，自动定位到文件末尾
    autocmd BufNewFile * normal G
endfunc

两件小事 2014-03-29

第一件事情是一个小意外

    今天和 Dragon 一起去看莫奈展，看完后回来遇到了一个小意外。Dragon小区楼下有一坨挖地挖出来的泥巴，他掉头时候没注意，不小心两个前轮陷入进去，不能前进也不能后退，卡住了。     我们折腾了一下午，又是用铁锹铲泥巴又是各种垫东西愣是没搞出来。后来隔壁一大叔很热心的来帮忙，一起用千斤顶顶起车子，把泥巴搞走，垫入硬板。还有其他几个路人一同帮助下终于把车子搞出来了。     中途，搞坏了一个千斤顶，有一个路人很臭屁认为自己是对的，最后帮了倒忙，然后甩手走了。但是很好的一点是，就算中途多次失败，来帮忙的大叔大婶们都没有停止帮助，一直帮到了最后。而且事儿完了后就挥手走人，深藏功与名，不图回报。     放一张完事后的自拍，都是泥巴，回来路上各种被让路，民工装太好用了!

第二件事儿是无责任传闻

    第二个就不是啥好事了，说的是一个道听途说来的小道消息。     话说国内某个互联网团队，因为团队经营不行，人都纷纷跑了。眼看团队要呵呵的节奏，某个小 Boss 居然开始动起离职员工的歪脑筋，开始调查离职员工的去向，意图使用一些手段来整这些跑路的人，纯属自己不爽也要拉你垫背的自杀式行为。     据内部不可靠消息称，这人一大爱好就是贬低比自己混的好的人。大抵是想通过让别人不爽来让自己爽。真是正事儿不干，乌七糟八的事儿搞了一大堆。

故事讲完了来个总结

    我觉得人生在世吧，也就那么回事儿，有空多帮助帮助别人，就像第一个故事里面的热心大叔一样。
    就算懒得管闲事也可以多想想怎么提高自己的知识水平，再不济看看金瓶梅苍老师什么的提高下姿势水平至少也造福大于等于一人吧。别老把心用在怎么折腾别人，怎么让别人不顺畅上面。先做人，后做事。说说很简单，但说了自个儿要先做到。     另外就是一点自勉，要虚怀若谷，别以为自己是个什么东西，其实也不过是一个井底之蛙，大不了就是大一点的井罢了。像第一个故事里面帮倒忙的路人一样，一上来就吼，老子是修车的，老子最懂，然后悲剧了。既然在井底，就多想想怎么看到更大的世界，而不是去别的青蛙的井里丢两块石头，这种一叶障目的想法，幼稚的不行，结果不仅仅没砸死别的青蛙，还把自己微弱的情商智商暴露无遗，损人不利己，何必呢。

那些一无所知的岁月，才是最美好的时光——题记

 傍晚的阳光，透过教学楼最高层钟楼的窗户犹如轻纱一般洒落在易澈的脸上，易澈眯起了双眼却继续看着下面熙熙攘攘的放学人流。  放学后的半小时，是易澈在上学日子里最喜欢的岁月，他并不喜欢匆匆的离开，在人挤人的状态下赶上公车，颠簸颠簸的到家。易澈喜欢观察人群离开的时刻，这个习惯不知道何时形成，却一直保留了下来，风雨无阻。每当放学时分，易澈总喜欢跑到学习教学楼最高层的钟楼里，趴在窗口看着楼下离开的人。有时候还会惊扰到偷偷在这里亲吻的稚嫩情侣，易澈也不在意，依然自顾自的看着。最后总是那些个情侣们受不了，离开这里，还给易澈一片亲近，易澈依然不懂声色的看着，似乎这是他本来的领地。  今儿似乎不是一个平常的日子，除了易澈外难得多了一个不是成双成对来这里的人。她是一个人静静的坐在一边，看着书，似乎在等待，但却从来不往窗外看上一眼。易澈盯着她看了一会儿，她也不动声色的继续看着书。“大抵是偶尔无出去来这里等家长的学霸吧。”易澈心里这样想着。然后又投入到了自己津津有味的观察人群的活动中去。  如果从科研角度来说，易澈实在算不上一个好的观察者，易澈从来不记录任何观察的现象。甚至昨天看到一个人很搞笑的摔的狗吃屎今儿也就忘记的一干二净。易澈自己觉得这样可以保持新鲜感，每天看似熙熙攘攘退去的人群，又混乱，又整齐，总让他着迷不堪。  有趣的是，那个看书姑娘每天都和易澈在钟楼相遇。一个看人，一个看书，互不打扰，也倒是一个风景线。当然这种互不打扰的状态带来了一个好处，那就是没有情侣再来骚扰易澈了，因为许多人看到有两个人影在钟楼，大多都会自叹来晚了然后默默离开。  易澈有时候也不看离开的人群，反而去看看那个姑娘手里的书。姑娘看书很快，几天就会换上一本。从卡夫卡的《变形记》到江户川乱步的《黄金假面人》，姑娘也算是品位独特。有一天，大抵是易澈盯着看太久了，姑娘收起了书，抬起头看着易澈问：

你也要一起看么？

 易澈愣了下，然后点了点头。姑娘没有多言，继续打开书，然后向右边挪了挪，让易澈在她左边坐下。两人便一起看起书来。那天看的是一本乱步的长篇——《怪人二十面相》，明智的小五郎和怪人的故事深深的吸引了两人。直到楼下离去的人群散尽，易澈都没有发现，姑娘倒是收起了书然后和易澈点点头告别。  自那以后，一个看人一个看书的状态，变成了一个看书，另一个也看书的状态，日复一日。
 后来易澈毕业，上大学，再也没有遇到过那个和他一同看书的姑娘，他也不曾问过姑娘的姓名信息。虽然那么小一个学校，其实只要耐心找找很快就能知道对方的姓名联系方式。但易澈就一直没有那么做，就像他一直没有记录每天观察人群离去的数据一样。大抵在他心中，这样就好，这样就好。  易澈偶尔还是会跑去一个高高的地方，看着人群流动，离去，归来。

写给自己的童话 2014-03-09

  七八年前，高中时代，那时候寝室还没有热水器，洗澡洗头很麻烦。男生倒还好，大不了冷水冲一下，可长头发的女生往往需要洗头，于是这洗头业务便养肥了学校周边的一大批理发店。
 那时候早恋的小青年们，不得不经历的一个事情就是陪女朋友去学校边上的理发店洗头。那是一个夏末秋至的时刻，天气依然炎热，但傍晚时分已经有了一丝清凉。学校后门不远处的一家理发店二楼，哦，或许说是阁楼更为合适，有一个少女正在享受洗头的服务。边上沙发上，慵懒的坐着一个年岁相仿的看书少年，这位少年时不时的会瞟一眼少女然后继续回到他手中的科幻世界中。

“是你男朋友吧。”洗发小妹突然出声，没等两人回答，小妹继续说道：“长得挺高大的。”

 少年的杂志微微的侧了一点，嘴角划出了一个细微的弧度，然后翻页，继续看书。

 这是我高中时代的故事，我甚至都已经不大记得当时人们的模样，但我却总记得这些儿琐碎而细小的事情。就像曾几何时，我特别痴迷嘴唇略略上扬的一个弧度，导致每次看到有人做如此表情时都会不由自主失神片刻。对比现在的自我，那时候的我真是卯足了劲用全身的感官在观察和感悟着这个世界呐。

我感觉那时候世界是彩色的，现在也还是彩色的，不过感觉灰度高了一些。

 这是我自己对自己说的话。我总觉得生活是一个很残忍的过程，从无限可能为开端，让你不停的选择，直到毫无可能为止。
 就像现在，我已经几乎不拍照了。虽然内心还是无数次涌动着去拍照的欲望，Amazon 的收藏夹里大半都是相机相关的，但我确确实实不拍照了。没不是时间，也不是人的关系，只是单纯的做出了自己的选择，对于摄影，我是觉得如果我不能像投入到编程那么多时间投入进去是不可能有所收获的。我不愿意只做一个爱好者，业余时间爬个山拍个日出，约个妞拍个写真，了了后期几笔便欣然自得以摄影师自居。如果达不到自己的极限，不如选择做一个实实在在的门外汉，只远观而不亵玩也。比起拍几张让亲朋好友称道的照片，我更希望能去了解光影的语言，去了解亚当斯区域曝光的原理，去了解每一种光线带来的改变，去分析每一个场景的光场，这才是我想要的摄影。

 所以，无论现在女神怎么说我，我都不愿意轻易拿起相机拍人。因为我知道，我对于摄影的无知已经让我停止了前进的脚步，所以与其拍一些自己都看不下去的东西，不如选择不产生。

 是非。
 这两个字基本上囊括了所有的生活。虽然许多事情对于整体来看，大家都觉得是中立而没有对错的。但对于个体的每一人，其实内心早有了是非的判断。  韭菜，够中立吧，有的人喜欢的不行，有的人厌恶的不行，就算对韭菜可有可无的人也有那么一点偏向。  不过生活就是因为没有一板一眼的剧本才显得丰富多彩。我想这才是我之后应该多观察的地方。  从小到大，我都是一个充满想象力的人。其实说白了，就是天生屌丝YY强。我厌恶跑步。每次长跑考试都跑的最后，难免会有一些无趣的人当作趣味围观，也有一些有趣的人为我呐喊加油。或许是觉得我是弱者，或许是觉得这是个笑话。不过我并不在意，每当我跑在最后的时候，虽然气喘吁吁难受不已，虽然被一些奇奇怪怪的人围观议论，我总是YY自己其实是露鸟狂奔的少年，那些人只是在羡慕鸟大而已。然后我就带着一脸傻逼一样幸福的表情，留着鼻涕被人套圈跑完全程。

 保持无知，保持幼稚。写给正在不知所措狂奔的露鸟少年。
