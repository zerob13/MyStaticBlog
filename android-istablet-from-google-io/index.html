Android 应用如何判断是否是平板（二）

2013-12-03

之前写过一篇 Android 判断平板手机的代码，觉得已经写的很飘逸了，而且挺符合规范的。不过还是需要同时修改 XML 和 Java 总觉得差了点什么。

最近阅读 Google I/O 的代码，看到里面有个判断方法，特别棒，所以就摘了出来，既然官方敢用估计还是比较准的，妥妥儿的用吧，代码和源代码链接如下~

具体文件点这里(UIUtils.java)

public static boolean isTablet(Context context) {
        return (context.getResources().getConfiguration().screenLayout
                & Configuration.SCREENLAYOUT_SIZE_MASK)
                >= Configuration.SCREENLAYOUT_SIZE_LARGE;
    }

查了下 SDK 文档对于 Configuration.SCREENLAYOUT_SIZE_LARGE 的解释是这样的：

Constant for screenLayout: a SCREENLAYOUT_SIZE_MASK value indicating the screen is at least approximately 480x640 dp units, corresponds to the large resource qualifier. See Supporting Multiple Screens for more information.

所以其实还是根据分辨率。
