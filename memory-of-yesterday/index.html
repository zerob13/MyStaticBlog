纪念昨天君

死胖子你最近很久没写东西了啊。

是啊，很久没写了。
死胖子有想法写点啥么？ 大抵需要一点仰望星空的灵感吧。 负分滚粗。

    好久没有写点东西了，再不写对不起服务器的租金，对不起域名的租金，对不起那几十个的订阅用户了。哦，其实只有十几个。     坐车上下班的时候，体内常常有声音响起，督促我是否该写点什么了。我想大抵我是应该写点什么了，但却又不知从何说起，说点什么。正能量？太无趣，鸡汤虽好，容易尿酸高。负能量？太简单，我的脸就是充满了负能量的喜感，贴个自拍就完事了。点评点科技圈？太麻烦，而且这方面的观点和天涯骂战一样，谁也不服谁，就像我不喜欢老罗，但我挺喜欢锤子，我既没有跑去喷罗胖子，同时也默默的订了一台锤子的 T1 。这样就好，这样就好。     既然扯倒了锤子，这算是我今年看下来比较又欲望买的东西了。我喜欢玻璃比例大的手机，比如 Nexus 4 ，现在我还在用，非常喜欢。另外，捐钱给 OpenSSL 这事儿做的漂亮，就冲着这个也应该买个他们的手机，不管是否作秀，就算作秀也希望以后各种厂能多作秀几次。也算是推动人类计算机事业发展了。     最近看着自己的 GitHub 提交记录，惨不忍睹，空空如也。着实觉得虚度了许多光阴，但又不能为了好看而瞎提交一些东西。所以想想为了充实业余生活，还是做点别的事情。然后就跑去翻译了一些外文的技术文章，丢给伯乐在线的 Android 开发小组。倒也有了两篇上了，不由自己觉得让世界变得美好了一点。其中一篇比较有意思，是写一个自己组建的 Android 开发框架，用了一堆第三方的库，但是总体效果很不错，有兴趣的同学可以看一下，点这里。个人建议， SnappyDB 和 EventBus 这两个比较有意思一点，其他的两个风格化太厉害，我个人并不推荐。     人生在世，扯扯犊子乃是一大乐事，当然码农锻炼身体也是必不可少都。每周末都会去打打球，算是最近比较好的一个事情，后续有机会还可以拉几个基友游泳，随缘。 最后送上逗逼视频一段，猛击这里 查看。
